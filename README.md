password-family
===============

在线密码管理，password manage on line.

## 　要求

* ruby 1.9.3
* rails 3.2
* mongoid 

### install

```ruby
git clone git@github.com:Byronlee/password-family.git
```

then bundle

```ruby
bundle
```
