# password-family  
you can manage all kinds of passwords with it online . 

please preview [test version](http://passwordfamily.ginchenorlee.com) and give me you advise.


# Requirements

* ruby 1.9.3
* rails 3.2
* mongodb & mongoid 

# install

```ruby
git clone git@github.com:Byronlee/password-family.git

bundle && rails s
```
